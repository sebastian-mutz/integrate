Adiabatic Lapse Rate (Dry): Example¶

\[\begin{equation} \Gamma = \frac {g}{C_p} = \frac { 9.81 ms^{-2} }{ 1006 Jkg^{-1}K^{-1} } = 9.75 Km^{-1} \end{equation}\]
