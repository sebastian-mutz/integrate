Adiabatic Lapse Rate
--------------------


Adiabatic Lapse Rate (Dry)
..........................
   
.. math:: 

    \begin{equation}
      \Delta U = Q - W
    \end{equation}      

.. math:: 

    \begin{equation}
      \Delta P = -\rho · \Delta z · g    
    \end{equation}          
    
.. math:: 

    \begin{equation}
      \Gamma = \frac {g}{C_p}   
    \end{equation}          
    

Adiabatic Lapse Rate (Dry): Example
...................................
   
.. math:: 

    \begin{equation}
      \Gamma = \frac {g}{C_p} 
      = \frac { 9.81 ms^{-2} }{ 1006 Jkg^{-1}K^{-1} }
      = 9.75 Km^{-1}
    \end{equation}
